# Cover Letter — Projection Mapping

An interactive cover letter where you drag a patch of light onto a building
to reveal the letter — a small nod to the projection-mapping work I want to do
and an ode to my current attempts to learn TD.

**How it was built:** scaffolded with AI assistance (the same way I'd start
a TouchDesigner or Resolume patch from a template), then customized by hand —
palette, animation timing, layout, copy, and the fiddly CSS/SVG tuning are mine.
I'm comfortable reading and modifying code like this; I'm not yet fluent enough
to write it from a blank file. Working on it ;)
